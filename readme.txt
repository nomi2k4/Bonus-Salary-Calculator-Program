Bonus Salary Calculator Program:
This C++ program was written to calculate the total salary including bonus based on the employee's grade.

Program Overview:

- Prompts the user to input their salary and grade (5-20)
- Calculates the bonus based on the grade:
    - Grade 15-20: 50% of salary
    - Grade 5-15: 25% of salary
- Displays the total salary including bonus

Features:

- User input for salary and grade
- Use of if-else statements for grade-based bonus calculation
- Clear and informative output

Assumptions:

- Bonus structure:
    - Grade 15-20: 50% of salary
    - Grade 5-15: 25% of salary

How to Run the Program:

1. Copy the provided C++ code into a .cpp file
2. Compile the program using a C++ compiler
3. Run the compiled program:
    - On Linux/macOS: ./bonus_salary_calculator
    - On Windows: bonus_salary_calculator.exe
4. Enter your salary and grade when prompted

Learning Objectives:

- Understanding basic user input and output
- Using if-else statements for conditional logic
- Working with floating-point variables and arithmetic operators
- Basic program structure and syntax
